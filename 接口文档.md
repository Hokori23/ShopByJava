# /user

## 注册【√】

**简要描述：** 

- 注册接口

**请求URL：** 
- ` /user`

**请求方式：**

- POST

**参数：** 

| 参数名   | 必选   | 类型   | 说明 |
| :------- | :----- | :----- | ---- |
| id       | **是** | String | 账号 |
| name     | **是** | String | 昵称 |
| password | **是** | String | 密码 |

**errcode：**

| 状态码 | 备注       |
| ------ | ---------- |
| 2      | 账号已存在 |

 **返回示例**

``` 
  {
    "errcode": 0,
    "data": "注册成功"
  }
```
